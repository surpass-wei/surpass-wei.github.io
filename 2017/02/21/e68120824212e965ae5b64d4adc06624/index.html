Spring Boot使用FastJson解析JSON数据

发表于 2017-02-21 | 分类于 开发

1.引入fastjson依赖库：


1.  `<!--添加fastjson解析JSON数据-->`
2.  `<dependency>`
3.  `    <groupId>com.alibaba</groupId>`
4.  `    <artifactId>fastjson</artifactId>`
5.  `    <version>1.2.16</version>`
6.  `</dependency>`

2.配置fastjson

这里要说下很重要的话，官方文档说的1.2.10以后，会有两个方法支持HttpMessageconvert，一个是FastJsonHttpMessageConverter，支持4.2以下的版本，一个是FastJsonHttpMessageConverter4支持4.2以上的版本，具体有什么区别暂时没有深入研究。这里也就是说：低版本的就不支持了，所以这里最低要求就是1.2.10+

方式一：

  • （1）启动类继承WebMvcConfigurerAdapter

  • （2）覆盖方法configureMessageConverters

具体代码：


1.  `@SpringBootApplication  //  申明让spring boot自动给程序进行必要的配置，等价于以默认属性使用@Configuration，@EnableAutoConfiguration和@ComponentScan`
2.  `public class Application extends WebMvcConfigurerAdapter{`
3.4.  `    @Override`
5.  `    public void configureMessageConverters(List<HttpMessageConverter<?>> converters) {`
6.  `        super.configureMessageConverters(converters);`
7.8.  `        //  初始化转换器`
9.  `        FastJsonHttpMessageConverter fastConvert = new FastJsonHttpMessageConverter();`
10.  `        //  初始化一个转换器配置`
11.  `        FastJsonConfig fastJsonConfig = new FastJsonConfig();`
12.  `        fastJsonConfig.setSerializerFeatures(SerializerFeature.PrettyFormat);`
13.  `        //  将配置设置给转换器并添加到HttpMessageConverter转换器列表中`
14.  `        fastConvert.setFastJsonConfig(fastJsonConfig);`
15.16.  `        converters.add(fastConvert);`
17.  `    }`
18.19.  `    public static void main(String[] args) {`
20.  `        SpringApplication.run(Application.class, args);`
21.  `    }`
22.  `}`

方式二：

  • 在配置类或启动类中，注入Bean : HttpMessageConverters

1.  `/**`
2.  ` * Bean配置管理`
3.  ` * Created by surpass.wei@gmail.com on 2017/2/21.`
4.  ` */`
5.  `@Configuration`
6.  `public class BeanConfig {`
7.8.  `    /*注入Bean : HttpMessageConverters，以支持fastjson*/`
9.  `    @Bean`
10.  `    public HttpMessageConverters fastJsonHttpMessageConverters() {`
11.  `        FastJsonHttpMessageConverter fastConvert = new FastJsonHttpMessageConverter();`
12.  `        FastJsonConfig fastJsonConfig = new FastJsonConfig();`
13.  `        fastJsonConfig.setSerializerFeatures(SerializerFeature.PrettyFormat);`
14.  `        fastConvert.setFastJsonConfig(fastJsonConfig);`
15.  `        return new HttpMessageConverters((HttpMessageConverter<?>) fastConvert);`
16.  `    }`
17.  `}`

配置完成后，在实体类中使用@JSONField(serialize=false)，是不是此字段就不返回了，如果是的话，那么恭喜你配置成功了，其中JSONField的包路径是：com.alibaba.fastjson.annotation.JSONField
