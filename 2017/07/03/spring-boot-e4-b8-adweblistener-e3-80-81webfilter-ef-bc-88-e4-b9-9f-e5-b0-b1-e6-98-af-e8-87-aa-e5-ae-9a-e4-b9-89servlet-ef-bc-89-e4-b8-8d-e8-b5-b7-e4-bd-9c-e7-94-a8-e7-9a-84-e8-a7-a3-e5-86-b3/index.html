spring boot中@WebListener、@WebFilter（也就是自定义servlet）不起作用的解决方案

发表于 2017-07-03 | 分类于 开发

在Spring Boot中，当需要以注解的方式添加session监听器或其他servlet(包括监听器、过滤器、以及自定义的servlet)时，为了让其生效，还需要在配置类上添加@ServletComponentScan注解，该注解的作用是告诉spring boot,让其自动管理servlet，否则注解方式注册的servlet不起作用

  1. 定义过滤器

1.  `import org.slf4j.Logger;`
2.  `import org.slf4j.LoggerFactory;`
3.4.  `import javax.servlet.*;`
5.  `import javax.servlet.annotation.WebFilter;`
6.  `import java.io.IOException;`
7.8.  `/**`
9.  ` * 使用注解标注过滤器@WebFilter将一个实现了javax.servlet.Filter接口的类定义为过滤器`
10.  ` * filterName声明过滤器的名称（可选）`
11.  ` * urlPatterns指定要过滤的URL模式,也可使用属性value来声明.(指定要过滤的URL模式是必选属性)`
12.  ` * <p>`
13.  ` * Created by surpass.wei@gmail.com on 2017/2/24.`
14.  ` */`
15.  `@WebFilter(filterName = "myFilter", urlPatterns = "/*")`
16.  `public class MyFilter implements Filter {`
17.  `    private Logger logger = LoggerFactory.getLogger(MyFilter.class);`
18.19.  `    @Override`
20.  `    public void init(FilterConfig filterConfig) throws ServletException {`
21.  `        logger.info("过滤器：myFilter >>>> 初始化");`
22.  `    }`
23.24.  `    @Override`
25.  `    public void doFilter(ServletRequest servletRequest, ServletResponse servletResponse, FilterChain filterChain) throws IOException, ServletException {`
26.  `        logger.info("过滤器：myFilter >>>> 执行过滤操作");`
27.  `        filterChain.doFilter(servletRequest, servletResponse);`
28.  `    }`
29.30.  `    @Override`
31.  `    public void destroy() {`
32.  `        logger.info("过滤器：myFilter >>>> 销毁");`
33.  `    }`
34.  `}`
  1. 定义监听器

1.  `import org.slf4j.Logger;`
2.  `import org.slf4j.LoggerFactory;`
3.4.  `import javax.servlet.annotation.WebListener;`
5.  `import javax.servlet.http.HttpSessionEvent;`
6.  `import javax.servlet.http.HttpSessionListener;`
7.8.  `/**`
9.  ` * HttpSession监听器`
10.  ` * <p>`
11.  ` * Created by surpass.wei@gmail.com on 2017/2/24.`
12.  ` */`
13.  `@WebListener`
14.  `public class MyHttpSessionListener implements HttpSessionListener {`
15.  `    private Logger logger = LoggerFactory.getLogger(MyHttpSessionListener.class);`
16.17.  `    @Override`
18.  `    public void sessionCreated(HttpSessionEvent httpSessionEvent) {`
19.  `        logger.info("监听器：Session >>>> 创建");`
20.  `    }`
21.22.  `    @Override`
23.  `    public void sessionDestroyed(HttpSessionEvent httpSessionEvent) {`
24.  `        logger.info("监听器：Session >>>> 销毁");`
25.  `    }`
26.  `}`
  1. 自定义servlet

1.  `import org.slf4j.Logger;`
2.  `import org.slf4j.LoggerFactory;`
3.4.  `import javax.servlet.ServletException;`
5.  `import javax.servlet.http.HttpServlet;`
6.  `import javax.servlet.http.HttpServletRequest;`
7.  `import javax.servlet.http.HttpServletResponse;`
8.  `import java.io.IOException;`
9.  `import java.io.PrintWriter;`
10.11.  `/**`
12.  ` * 自定义Servlet`
13.  ` * <p>`
14.  ` * Created by surpass.wei@gmail.com on 2017/2/24.`
15.  ` */`
16.  `public class MyServlet1 extends HttpServlet {`
17.  `    private Logger logger = LoggerFactory.getLogger(MyServlet1.class);`
18.19.  `    @Override`
20.  `    protected void doGet(HttpServletRequest req, HttpServletResponse resp) throws ServletException, IOException {`
21.  `        logger.info("已执行：MyServlet1.doGet()");`
22.  `        doPost(req, resp);`
23.  `    }`
24.25.  `    @Override`
26.  `    protected void doPost(HttpServletRequest req, HttpServletResponse resp) throws ServletException, IOException {`
27.  `        logger.info("已执行：MyServlet1.doPost()");`
28.  `        resp.setContentType("text/html");`
29.  `        resp.setCharacterEncoding("UTF-8"); //  设置编码，解决中文乱码问题`
30.31.  `        PrintWriter out = resp.getWriter();`
32.  `        out.println("<html>");`
33.  `        out.println("<head>");`
34.  `        out.println("<title>Hello World</title>");`
35.  `        out.println("</head>");`
36.  `        out.println("<body>");`
37.  `        out.println("<h1>这是：MyServlet1</h1>");`
38.  `        out.println("</body>");`
39.  `        out.println("</html>");`
40.  `    }`
41.  `}`

配置@ServletComponentScan以使以上servlet生效


1.  `@SpringBootApplication`
2.  `@ServletComponentScan`
3.  `public class BasicApp extends WebMvcConfigurerAdapter{`
4.5.  `    @Bean`
6.  `    public ServletRegistrationBean myServlet1() {`
7.  `        return new ServletRegistrationBean(new MyServlet1(),"/myServlet1/*");`
8.  `    }`
9.10.  `    public static void main(String[] args) {`
11.  `        SpringApplication.run(BasicApp.class, args);`
12.  `    }`
13.  `}`
